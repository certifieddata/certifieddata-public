CertifiedData.io — Script Tag Install

Add a verification badge to any webpage with a single <script> tag. Replace YOUR_CERT_ID with a real certificate UUID.

Compact badge

Live demo: 
<script
  src="https://certifieddata.io/embed.js"
  data-cert-id="YOUR_CERT_ID"
  data-variant="compact"
  data-theme="dark">
</script>

Inline badge (default)

Live demo: 
<script
  src="https://certifieddata.io/embed.js"
  data-cert-id="YOUR_CERT_ID"
  data-variant="inline"
  data-theme="dark">
</script>

Full trust card

Live demo: 
<script
  src="https://certifieddata.io/embed.js"
  data-cert-id="YOUR_CERT_ID"
  data-variant="full"
  data-theme="dark"
  data-show-hash="true">
</script>

Light theme (for light-background pages)

Live demo: 
<script
  src="https://certifieddata.io/embed.js"
  data-cert-id="YOUR_CERT_ID"
  data-variant="inline"
  data-theme="light">
</script>

With machine-readable JSON-LD injection

<script
  src="https://certifieddata.io/embed.js"
  data-cert-id="YOUR_CERT_ID"
  data-variant="inline"
  data-show-machine-links="true">
</script>

All supported data attributes

data-cert-id            (required)  Certificate UUID
data-variant            compact | inline | full    (default: inline)
data-theme              light | dark | auto         (default: dark)
data-show-hash          true | false                (default: false)
data-link-target        _blank | _self              (default: _blank)
data-show-machine-links true | false                (default: false)
