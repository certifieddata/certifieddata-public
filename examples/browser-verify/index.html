Certificate Verification

Cryptographically verify a CertifiedData certificate — runs entirely in your browser using the Web Crypto API. No data leaves your machine except to fetch the public manifest and signing keys from certifieddata.io.

This page performs independent Ed25519 signature verification using the Web Crypto API (SubtleCrypto.verify). The signed manifest is fetched from certifieddata.io along with the public signing key. The signature is verified locally — no trust in the server response is required.
